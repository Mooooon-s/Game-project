This package contains the example scenes and assets required to test the CGLocalization tool in
the Editor and other platforms.

To open the example scenes go to the 'CGLocalization > Examples > Scenes' folder.
Two scenes are included:
    1. Localization Example: contains a small example with a basic main menu and options panel.
    2. Localized Behaviours: contains an example showing the localized behaviours.
 
NOTES:
1. These examples use the runtime .dll file of the version 1.0 (From the version 1.1
   and on the source code is included).
2. If you purchase the full package, delete this package completelly. The full version
   also contains these examples.
